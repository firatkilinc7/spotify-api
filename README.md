# Spotify Api

<li>Getting Started</li>

        composer update
    
        php artisan key:generate
    
        php artisan migrate
        
        php artisan passport:install

<br>
<li>Swagger can be available:</li>
    
        url      : /api/documentation
        email    : test@test.com
        password : 12345678

</li>

<li>After receiving your token log in via Swagger.

        Bearer TOKEN_ADRESS

</li>
